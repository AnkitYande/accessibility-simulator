Take Aways

Relevant WCAG success criterion are shown in parentheses


Dexterity

  • Ensure your website is operable through a keyboard interface (2.1.1)
  • Ensure focus of current element is visible when using a keyboard interface (2.4.7)
  • Ensure touchpoints such as buttons are large (2.5.5)

Vision

  • Ensure color is not the only visual means of conveying information (4.1.1)
  • Ensure contrast for text is above the minimum ratio (1.4.3/ 1.4.6)
  • Ensure font size can be increased up to 200 percent without loss of content or functionality (1.4.4)

Screen Readers

  • Ensure non-text content such as images have alt text (1.1.1)
  • Ensure the purpose and redirection of each link is clear (2.4.4)
  • Ensure proper HTML tags, headings, and labels are used (2.4.6)
  • Ensure proper labels are present for form inputs (3.3.2)
