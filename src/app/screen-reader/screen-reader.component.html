Behind this cover is a website that you can access using your keyboard and a built in screen reader. Using this screen reader try to complete the following task:


  1. Use the Down Arrow and Tab keys to get started
  2. Navigate through the website until you reach the table
  3. Once you reach the table, use the table commands to try to find which laptop is under $1000 and has at least 16GB of memory
  4. Navigate out of the table and down to the form
  5. Fill out the form using the Form controls. Select chose the model that meets the criteria in bullet three in snow silver
  6. Press the "Submit" button to finish the simulation

Submitting the form in the simulation will take you to the next page, or you can click here to skip.

You can operate the website using the following commands
(these are based on the NVDA screen reader )
General
Down Arrow Go to the next line
Up Arrow Go to the previous line
Tab Jump to the next focusable item (e.g. link, button, inputs)
Enter Select a link or button
Table
Alt/Option + Right Arrow Cell to the right
Alt/Option + Left Arrow Cell to the left
Alt/Option + Up Arrow Cell above
Alt/Option + Down Arrow Cell below
Forms
Right Arrow Select the next element in the form group
Left Arrow Select the previous element in the form group
Tab Jump out of fieldset to next focusable element
